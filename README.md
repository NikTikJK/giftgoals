# GiftGoals

Веб‑приложение социального вишлиста, в котором пользователь создаёт списки желаний (вишлисты), делится ими по ссылке, а друзья могут резервировать подарки и скидываться на дорогие позиции без раскрытия личности и сумм для владельца.

## Описание проекта

GiftGoals — это сервис, который решает два сценария:
- **Владелец** ведёт свои вишлисты (под ДР, НГ и любые события), добавляет подарки с ценой, ссылкой и картинкой.
- **Друзья** по публичной ссылке видят список, могут анонимно резервировать подарки и участвовать в общем сборе на «дорогие» подарки, не ломая сюрприз.

Логика и поведение системы подробно описаны в `spec/PRD.md`, UX/UI‑требования — в `spec/UX.md`, а пошаговый план реализации — в `spec/IMPLEMENTATION.md`.

## Технологический стек (план)

- **Фронтенд**: React SPA (Vite + React + TypeScript).
- **Бэкенд**: Express (Node.js + TypeScript), REST/JSON API.
- **База данных**: PostgreSQL (managed PaaS: Supabase/Railway/Render и т.п.).
